# ☕ Serenatto - Análise de Dados

## 📌 Sobre o Projeto

O **Serenatto** é um projeto de análise de dados com foco em extrair insights estratégicos a partir de um dataset de vendas. O objetivo é entender o comportamento dos clientes, identificar padrões de consumo e gerar informações úteis para tomada de decisão.

---

## 🎯 Objetivo

Transformar dados brutos em insights relevantes, respondendo perguntas como:

* Quais produtos são mais vendidos?
* Qual o ticket médio por cliente?
* Existe sazonalidade nas vendas?
* Métodos de pagamento influenciam o consumo?

---

## 📊 Etapa Atual

✔️ **Análise Exploratória de Dados (EDA) concluída**

Durante essa etapa foram realizadas:

* Limpeza e tratamento dos dados
* Análise de produtos mais vendidos
* Estudo dos métodos de pagamento
* Cálculo de receita total
* Cálculo de ticket médio por cliente
* Teste de hipóteses (ex: relação entre preço e volume de vendas)
